SHEET 03 — RESTAURANT OPERATIONS

Savora

FULL-STACK · MERN · RESTAURANT MANAGEMENT

A complete restaurant operations platform covering menu management, order tracking, table reservations, and role-based access for admins, staff, and customers.

The backend handles authentication via JWT and Google OAuth, with a MongoDB data layer behind a RESTful Express API. The frontend is a TypeScript React app using TanStack Router for client-side navigation.
